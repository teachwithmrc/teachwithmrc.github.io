# Meta / Facebook Ads Plan

Last updated: 2026-03-22

## Main recommendation

Record one master version of each generator demo in `1080 x 1920` vertical.

Then crop/export from that master into the other key ad sizes.

That gives you:

- Reels
- Stories
- vertical feed ads
- square feed ads

without re-recording the same demo multiple times.

## Best ad sizes to make

These are the only sizes you really need for generator demo ads.

### 1. Vertical master

- `1080 x 1920`
- ratio: `9:16`
- use for: Facebook Reels, Instagram Reels, Facebook Stories, Instagram Stories

### 2. Feed vertical

- `1080 x 1350`
- ratio: `4:5`
- use for: Facebook Feed, Instagram Feed

### 3. Square backup

- `1080 x 1080`
- ratio: `1:1`
- use for: Feed placements when you want a simpler crop

### 4. Wide only if needed

- `1200 x 628`
- ratio: `1.91:1`
- use for: legacy link ad layouts or special placements

This should not be your priority for generator demos.

## Best production workflow

### Record once

Record the screen in `1080 x 1920`.

### Keep safe margins

Leave space:

- at the top for platform UI
- at the bottom for captions and CTA
- avoid putting tiny text right against the top or bottom edges

### Export three versions

For each strong demo, export:

- `9:16`
- `4:5`
- `1:1`

## Best practical recording setup for generator demos

### Screen recording style

- zoom in tightly on the generator controls
- make the cursor/finger movement obvious
- avoid showing tiny unreadable menus
- show the result page clearly

### Best clip structure

1. Hook
2. Show the problem
3. Show the generator
4. Show the result
5. CTA

### Best length

- `10–20 seconds` for cold ad creatives
- `20–30 seconds` for stronger demo ads
